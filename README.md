# SGV - ProWeb

Sistema de Gestão de Vendas

---

## 📘 Sobre o Projeto

Este projeto foi desenvolvido no âmbito da disciplina de **Programação Web**.

O objetivo é conceber e implementar um **Sistema de Gestão de Vendas (SGV)**, capaz de gerir produtos, clientes e processos de venda, evoluindo desde um site estático até uma aplicação web completa em produção.

---

## 🎯 Objetivo Geral

Desenvolver competências técnicas e arquitecturais para construir aplicações web completas — desde os fundamentos da web até ao deploy em produção.

---

## 🎯 Objetivos Específicos

* Compreender o funcionamento da web (HTTP, cliente/servidor)
* Criar interfaces com HTML5 semântico e CSS3
* Programar interatividade com JavaScript moderno (ES6+)
* Desenvolver APIs com Node.js e Express
* Trabalhar com bases de dados (SQL)
* Implementar autenticação (JWT)
* Utilizar React e Next.js
* Trabalhar em equipa com Git e GitHub

---

## 📅 Metodologia

O projeto é desenvolvido de forma progressiva ao longo do semestre, com entregas semanais e evolução contínua até um produto final funcional.

---

## 🔁 Fluxo de Trabalho (Git)

Este projeto segue um modelo profissional de colaboração:

1. Criar uma branch (`feature/nome-da-tarefa`)
2. Desenvolver a funcionalidade
3. Fazer commits semânticos
4. Fazer push para o GitHub
5. Abrir Pull Request
6. Code review por colegas
7. Merge para a branch `main`

---

## ✍️ Commits Semânticos

Formato obrigatório:

[AREA] descrição clara no imperativo

### Exemplos:

* [HTML] criar estrutura da página inicial
* [CSS] implementar layout responsivo
* [JS] validar formulário
* [API] criar endpoint de produtos
* [AUTH] implementar login com JWT

---

## ⚠️ Regras do Projeto

* ❌ Não fazer push directo para a `main`
* ✅ Usar sempre branches
* ✅ Fazer commits claros e descritivos
* ✅ Participar semanalmente

---

## 📊 Avaliação

A avaliação é baseada em:

* **Commits individuais (50%)**
* **Avaliação intercalar (20%)**
* **Apresentação final (20%)**
* **Auto-avaliação (10%)**

## 📦 Entregas
